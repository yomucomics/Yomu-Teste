## 📋 Opções do Teste

### Opção 1 — Web (React / Next.js)

Crie uma pequena aplicação web que exiba uma lista de **manhwas**.

#### ✅ Requisitos

- Usar **React** ou **Next.js**
- Usar **TypeScript**
- Exibir uma lista de manhwas contendo:
  - 🖼️ Capa
  - 📖 Título
  - 🔖 Status (`ongoing` / `completed`)

#### ⚡ Funcionalidades

- [ ] Listagem de manhwas
- [ ] Página de detalhes de cada manhwa
- [ ] Botão para **favoritar / desfavoritar**

---

### Opção 2 — Mobile (React Native / Expo)

Crie um aplicativo simples que exiba uma lista de **manhwas**.

#### ✅ Requisitos

- Usar **React Native** ou **Expo**
- Usar **TypeScript**

#### ⚡ Funcionalidades

- [ ] Tela de lista de manhwas
- [ ] Tela de detalhes de cada manhwa
- [ ] Botão para **favoritar**

## 🚀 Como entregar

1. Suba o projeto em um repositório **público no GitHub**
2. Inclua um `README.md` no projeto com:
   - Instruções para rodar localmente
   - Qual opção você escolheu e por quê
   - Qualquer observação que achar relevante
3. Envie o link do repositório
4. Depois de terminar, entre no **Discord da Yomu** e abra um ticket enviando o link do projeto para avaliação:  
   https://discord.gg/4F4YKXXB7S
